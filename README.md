# 🛒 React E-commerce MT

Una demostración interactiva de comercio electrónico construida como una aplicación. Este proyecto destaca por un enfoque técnico ultraligero que implementa React.

## Características Principales

*   **Página de Detalles del Producto (PDP):** Al interactuar con el catálogo.
*   **Gestión Global de Estado (Local State):** Uso de `useState` para gobernar independientemente un sistema de Carrito de Compras, permitiendo añadir, restar cantidades y ver subtotales en vivo.
*   **Manejo Inteligente del DOM:** Se implementaron controles lógicos que reaccionan a variables, como el ocultamiento automático del selector de Tallas para productos que no entran en la categoría de indumentaria ("ropa").
*   **Formateo Localizado (COP):** Los precios son manipulados por el iterador de JavaScript local para transformarse e imprimirse estéticamente bajo la moneda del peso colombiano (COP).

## Tecnologías Utilizadas

*   **React 18** (Vía UMD/CDN)
*   **Babel Standalone** (Para traducción JSX a JS nativo)
*   **HTML5**
*   **CSS3** (CSS Grid, Flexbox, y Variables nativas para consistencia de tema).
*   **JavaScript**
